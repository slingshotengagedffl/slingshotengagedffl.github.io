How to publish your note
  1. Set the Week and Year, then write your note using the toolbar for formatting.
  2. Click Copy to Clipboard.
  3. Go to your GitHub repo and click data.js.
  4. Click the pencil icon, select all the existing text, and paste.
  5. Click Commit changes — your site updates in about 30 seconds.
